# Headless business component

A collection of common business components. Simply copy the complete source code into your project to run it.

## 说明

总结工作中出现的一些组件开发其中一些是对第三方组件库的 hack 或者是二次开发(也许是最佳实践)
也会持续增加一些开发莫个功能时用到的最佳组件选择(第三方) 当然如果你的美工对用户 UI/UX 有极高的要求那么使用此代码就再合适不过了 你可以直接拷贝到你的项目中仅需安装一些第三方依赖大部分都是需深度二开的 当然有些复杂的业务组件是直接纯自研的给你提供一些实现的思路 常见的一些动效可能没有参考做起来会费时费力可以减少一些你的工作量
