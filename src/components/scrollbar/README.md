# Vue Custom Scrollbar Component

Extract the scrollbar component from the [element-plus](https://element-plus.org/zh-CN/component/scrollbar) library and optimize its performance.

## Features

- Zero dependencies
- A pure component with source code that can be copied directly
- Maybe the performance will be a little better
- Supports all [element-plus](https://element-plus.org/zh-CN/component/scrollbar) scrollbar component properties

If your project doesn't need the entire element-plus component library and you only need a custom scrollbar component, you might want to give this a try.
